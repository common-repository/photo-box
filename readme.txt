== Description ==   

The Photo Box will set popup for all images in galleries.

Auto popup in home page, slide show popup, responsive popup for gallery in posts, pages, ...

You can use do_shortcode('[gallery type="photobox" ids="1,2"]').
  
Support to block gallery - Gutenberg editor.

[Demo plugin](http://photoboxone.com/photo-box-plugin/).  

= The plugin allows using =  

* Gallery
* Slide show
* Auto popup
* Popup
* Popup single 
* Popup slider  
* Block gallery - Gutenberg editor 

[How to use Photo Box plugin](http://photoboxone.com/how-to-use-photo-box).

== Installation ==
  
#### Thank you for your interest in Photo Box plugin.
### Minimum requirements.

*  Wordpress 4.0+ 
*  PHP 5.x
*  MySQL 5.x.
This section describes how to install the [plugin wordpress](http://photoboxone.com/how-to-install-plugin-in-wordpress/) and get it working.
e.g.

1. Upload folder 'plugin-name' to the `/wp-content/plugins/` directory. 
2. Activate the "Photo Box" [plugin](http://photoboxone.com/how-to-use-photo-box) through the '[Plugins](http://photoboxone.com/category/plugins/)' menu in the Wordpress.
3. Go to edit [gallery](http://photoboxone.com/photo-box-plugin/) in media popup and choose type is "Photo Box" and setting.

If any problem occurs, please read the [documents Wordpress](http://photoboxone.com/category/documents/).
